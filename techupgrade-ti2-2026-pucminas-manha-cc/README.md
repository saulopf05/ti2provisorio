## Instruções de utilização

A Home Page do TechUpgrade funciona como o portal central de informações, onde você compreende a proposta de valor do serviço, consulta as perguntas frequentes e visualiza os benefícios de utilizar a plataforma, como o acesso a recomendações confiáveis e links diretos para compra. A partir dela, você é direcionado para a Tela de Análise, o núcleo funcional do site. Nessa etapa, o fluxo é dividido entre o envio de uma imagem das configurações do seu computador e a escolha de um perfil de uso específico, como jogos, trabalho ou programação, garantindo que o diagnóstico seja personalizado para a sua necessidade real.

Após o processamento dessas informações, o sistema exibe a Tela de Resultados, que apresenta um diagnóstico completo do seu hardware. Nela, cada componente é classificado como adequado ou insuficiente, acompanhado de sugestões de peças ideais para o upgrade, valores médios de mercado e botões que levam diretamente às lojas parceiras. Por fim, o site oferece as Telas de Usuário, onde é possível realizar login ou criar uma nova conta para salvar o histórico de suas análises e manter um registro organizado de todas as recomendações recebidas.
